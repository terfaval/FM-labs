# FM-Lab

A collection of small, structured digital tools.

---

## About

FM-Lab is a portfolio of projects built from personal research.

Each project starts from exploring a topic, then turns into a system:
something that does not just store information, but makes it navigable and usable.

The focus is on:
- structure
- curation
- discoverability
- calm, readable interfaces

---

## Projects

The projects are defined in:

`/content/portfolio_content_pack.md`

The web app reads from this file and renders:
- introduction
- featured projects
- other projects
- individual project pages

---

## Tech

- Next.js (App Router)
- TypeScript
- Markdown-based content

No database.
No CMS.
No backend logic in v0.

---

## Philosophy

This is not a startup landing page.

The goal is to create a quiet, structured space
where projects can be explored without noise.
